# Source Offer For Bundled Git

LovelyGit release packages may redistribute executable Git binaries built by
`desktop/dugite-native`.

The complete corresponding source code and build tooling for the bundled Git
distribution are available from the upstream release used by this application:

```text
https://github.com/desktop/dugite-native/tree/v2.53.0-3
```

The exact upstream dependency manifest for that release is available from:

```text
https://raw.githubusercontent.com/desktop/dugite-native/v2.53.0-3/dependencies.json
```

If you received a LovelyGit release package that includes bundled Git binaries
and you cannot access the source at the URLs above, contact the LovelyGit
distributor for a copy of the corresponding source code. Any physical transfer
charge will be no more than the distributor's cost of performing source
distribution.

This offer is intended to remain valid for at least three years from the date of
the LovelyGit release that includes the bundled Git binaries.

If LovelyGit ever distributes modified Git binaries, the modified source code
and build scripts must also be published and referenced here.
